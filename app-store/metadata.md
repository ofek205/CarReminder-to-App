CarReminder היא האפליקציה הישראלית לניהול רכב — תזכורות חכמות, מסמכים דיגיטליים, מוסכים בקרבת מקום ומומחה AI שמכיר כל רכב. עובדת מצוין על מכוניות, אופנועים, משאיות, יאכטות, סירות וכלי שטח.

⏰ אל תפספסו שום תאריך
התראה לפני טסט, חידוש רישיון, פוליסת ביטוח, החלפת שמן, החלפת צמיגים, בלמים, מצבר, סיכה ועוד. האפליקציה לומדת את הרכב שלכם ומחשבת אוטומטית מתי הטיפול הבא — לפי קילומטראז׳, גיל הרכב והיסטוריית הטיפולים.

### Notes for Reviewer (4000 chars)
```
Thank you for reviewing CarReminder!

CarReminder is a Hebrew RTL vehicle management app for Israeli drivers, boat owners, motorcyclists and off-road enthusiasts. The interface is fully Hebrew but the codebase, error messages and developer logs are English.

GUEST MODE: You can explore the entire app without signing in by tapping "כניסה כאורח - בלי הרשמה" (Continue as Guest) on the welcome screen. The guest session is pre-seeded with a demo car and demo vessel so all features have data immediately.

DEMO ACCOUNT: For features that require an account (sharing, community posting), use the credentials above. The demo account is pre-populated with sample vehicles and documents. Real user data has been excluded.

LOCATION ACCESS: The "Find Garage" feature requests location to show nearby auto-shops. The data source is OpenStreetMap (public). If you deny location, the map falls back to Tel Aviv center.

CAMERA / PHOTO ACCESS: Used to scan vehicle license cards and documents. The OCR runs through Google Gemini API; only the photo bytes are sent, no personal identifiers.

NOTIFICATIONS: Local notifications only (Capacitor LocalNotifications plugin). No push servers. Test the notification engine via Settings → Reminders → "Send test notification".

USER-GENERATED CONTENT: The Community section has Report, Block, and admin moderation. Posts pass through a server-side sanitizer that strips HTML/scripts and rate-limits to 5 posts/min. Inappropriate content is removed within 24h.

Thank you!
```
