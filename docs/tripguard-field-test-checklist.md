## 1. הגדרה ראשונית (flow)
- [ ] **P0** חשבון **אישי** → הגדרות → התראות → מופיע "בטיחות ילדים"
- [ ] **P0** disclaimer: אי-אפשר להמשיך בלי לסמן את ה-checkbox
- [ ] **P0** מופיעה רשימת מכשירי ה-Bluetooth **המזווגים האמיתיים** (לא mock)
- [ ] רמז "נשמע כמו אוזניות" מופיע על AirPods/אוזניות
- [ ] **P0** אחרי בחירת רכב + הרשאות + BT דלוק → מחוון 🟢 "ההגנה פעילה"

## 2. Happy path אמיתי (הליבה)
- [ ] **P0** התחבר לרכב → סע ≥ 2 דק' → התנתק → **התראה רוטטת תוך ≤10ש'**
- [ ] **P0** ההתראה: "סיימת נסיעה / ודא שכל הילדים ירדו מהרכב" + רטט מורגש
- [ ] **P0** כפתור "בדקתי, הכל בסדר" → ההתראה נסגרת
- [ ] כפתור "אין ילדים ברכב" → נסגר
- [ ] לחיצה על גוף ההתראה → פותח את מסך "בטיחות ילדים"

## 3. עמידוּת ברקע (המבחן האמיתי) — חזור על כל אלה:
| תרחיש | Samsung | Xiaomi | Pixel |
|---|---|---|---|
| **P0** אפליקציה הרוגה (swipe-kill) → נסיעה → התראה נורית | ☐ | ☐ | ☐ |
| **P0** מסך נעול/כבוי → התראה + רטט מעירים | ☐ | ☐ | ☐ |
| **P0** אחרי reboot (בלי לפתוח אפליקציה) → נסיעה → התראה | ☐ | ☐ | ☐ |
- [ ] אם נכשל ב-OEM כלשהו → כפתור "פתח הגדרות סוללה" עובד + בטל אופטימיזציה + נסה שוב

## 4. escalation + לוג
- [ ] התעלם מההתראה 30ש' → **התראה חוזרת חזקה יותר** ("עדיין לא בדקת את הרכב")
- [ ] "בדקתי" לפני 30ש' → אין escalation
- [ ] "נסיעות אחרונות שזוהו" מציג את הנסיעות + "נשלחה תזכורת"/"ללא תזכורת"

## 5. תרחישי כשל (false-negative חמור מ-false-positive)
- [ ] **P0** BT כבוי → מחוון 🔴 "ה-Bluetooth כבוי" (לעולם לא ירוק כוזב)
- [ ] **P0** הרשאת BT/התראות נשללה → 🔴 + "תקן עכשיו" מבקש הרשאה אמיתית
- [ ] **P0** עצירת דלק (חיבור-ניתוק < 2 דק') → אין התראה
- [ ] דרופ רגעי של BT באמצע נסיעה → לא נספר כסוף-נסיעה (debounce)
- [ ] **P0** כיבוי ה-toggle הראשי → נסיעה → אין התראה + מחוון 🔴 כבוי
- [ ] **P0** אופטימיזציית סוללה דולקת → מחוון עדיין יכול 🟢 + הערה כתומה (advisory, לא חוסם)
- [ ] נסיעה שהתחילה בחלון-זמן ונגמרה מחוצה לו → **כן** מתריע
- [ ] מחוץ לחלון שעות/ימים → אין התראה
- [ ] כיול: 2 דק' לא מפספס נסיעות אמת ולא מתריע על עצירות קצרות _(לכוונן אם צריך)_
